# 🚀 Campus2Career

> AI-Powered Career Coach for Students

## 📌 Problem

Many students apply for internships and jobs without understanding why they get rejected.

Most resume analysis platforms either:

- Provide only an ATS score
- Hide important insights behind paywalls
- Do not offer actionable guidance

As a result, students struggle to identify skill gaps and prepare effectively for their dream roles.

---

## 💡 Solution

Campus2Career is an AI-powered career coach that analyzes a student's resume and provides personalized career guidance.

Using Google Gemini AI, the platform helps students understand:

- 🎯 Career Readiness Score
- 📈 Key Strengths
- ⚠️ Missing Skills
- 🚀 Recommended Projects
- 🗺️ Personalized Learning Roadmap

The goal is to provide meaningful career insights without hiding core recommendations behind a paywall.

---

## ✨ Features

### 🎯 Resume Analysis
Analyze resumes against a selected target role.

### 📊 Career Readiness Score
Get a score indicating readiness for a chosen career path.

### ⚠️ Skill Gap Detection
Identify missing skills required for the selected role.

### 🚀 Project Recommendations
Receive project ideas that strengthen your profile.

### 🗺️ Personalized Learning Roadmap
Get a step-by-step learning plan to become job-ready.

### 🤖 AI-Powered Insights
Powered by Google Gemini AI for intelligent recommendations.
